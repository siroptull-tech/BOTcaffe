# 🤖 BOTcaffe

Telegram-бот для автоматизации работы антикафе и коворкингов.

## 📋 О проекте
Бот решает главные задачи антикафе: бронирование столов, учёт времени посетителей и автоматический расчёт оплаты. Всё это через удобный интерфейс Telegram.

## 🚀 Функционал

### Для посетителей:
- 📅 Бронирование столов и комнат
- ⏱ Запуск и остановка таймера посещения
- 💰 Просмотр текущего счёта
- 📝 История посещений

### Для администраторов:
- 📊 Статистика посещаемости
- 👥 Управление бронированиями
- 💵 Настройка тарифов (почасовая оплата, пакеты часов)
- 📈 Отчёты по выручке

## 🛠 Технологии
- Python 3.x
- python-telegram-bot
- SQLite / PostgreSQL
- SQLAlchemy (для работы с БД)
